# Weather Bot
Запуск производится при помощи команды python3 bot.py
Бот, показывающий погоду по дням, по часам, а также штормовые предупреждения.
Команды бота:
/help - Список команд бота
/current_weather - Погода на данный момент
/hourly_weather - Погода в следующие 48 часов по часам
/alerts - Штормовые предупреждения
/daily_weather - Прогноз погоды на следующие 7 дней
/coordinates - Координаты города
